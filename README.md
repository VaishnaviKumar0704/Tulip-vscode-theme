# 🌷 Tulip Theme for Visual Studio Code

A soft, elegant, tulip-inspired theme for Visual Studio Code that brings calm floral tones and a refreshing aesthetic to your coding environment.

## ✨ Preview

A warm blend of pastel pinks, greens, and soft neutrals designed to reduce eye strain and make long coding sessions more enjoyable.

## 🎨 Features
🌷 Tulip-inspired color palette (soft pinks, greens, and creamy backgrounds)
🌙 Light and dark friendly variants
💻 Optimized syntax highlighting for readability
🔤 Clean typography focus
🌿 Minimal UI distractions
⚡ Smooth contrast for long coding sessions

## 📦 Installation
1. Download .vsix file
2. Open VS Code
3. Extensions -> Install from VSIX
